#This is an readme for the data_conversion.py file.
The data_conversion.py file is used to convert the numerical regression and continous datafile into the binary data file.
For using this file, import the data file. For this substitute the path of the file in the 3rd line of the code.
The input file to this program should be a file with the headers and data in the numerical and real integer format.
THe output of this program is a new file with binary data.
