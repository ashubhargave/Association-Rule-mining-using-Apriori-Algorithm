#This is an readme for the confidence.py file.
The confidence.py file is used to find the top 10 association rules using confidence and lift. frequent itemsets method.
For using this file, import the data file. For this substitute the path of the file in the 3rd line of the code.
The input file to this program should be a file with the headers and data in the bindary 0 and 1's format.
THe output of this file is the set of top 10 rules using lift and confidence interesting parameters. printed on the console as length of candidate frequent itermsets and frequent itemsets.
